要保存某个 **tmux 会话里所有的输出打印**（包括你之前已经看不到的历史内容），常用的方法有 3 种，适合不同场景👇：

---

## ✅ 方法 1：使用 `tmux capture-pane`（最推荐）

`tmux` 自带的命令可以直接把某个窗口/面板的**全部历史输出**保存到文件中：

```bash
tmux capture-pane -p -S - > output.log
```

解释：

* `-p`：打印到标准输出（可以用 `>` 重定向保存）
* `-S -`：从历史缓冲区的开头开始捕获（不加这个只会保存当前可见区域）

如果你在多个 pane 或窗口中运行，可以指定：

```bash
tmux capture-pane -t <session_name>:<window_index>.<pane_index> -p -S - > output.log
```

例子：

```bash
tmux capture-pane -t mysession:0.0 -p -S - > logs.txt
```

👉 优点：

* 能抓取整个缓冲区，包括**屏幕上看不到的历史内容**
* 不需要提前设置日志

👉 缺点：

* 只能手动导出一次（想实时保存看方法 2）
